        ● зелений — краще за бенчмарк сегмента; ● червоний — є простір для покращення. Колонка «Тренд» = зміна Q1→Q2 для бренду.
        Методологія. Джерело: агреговані показники Bolt Food (CVP). Бенчмарк — медіана по інших grocery-партнерах сегмента (без брендів групи), за той самий місяць/квартал; конкретні партнери не розкриваються. Квартали: Q1 = січень–березень, Q2 = квітень–червень. Потокові показники (GMV, замовлення, покази, перегляди меню, нові клієнти) підсумовуються за квартал; ставки та середні — усереднюються. Для кожного показника визначено природний напрям («більше краще» чи «менше краще»).
